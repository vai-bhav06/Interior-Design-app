# 🏡 Decorio - Interior Design Website

Decorio is a modern and responsive Interior Design website designed in **Figma** and developed using **React.js** and **Vite**. The project focuses on providing a clean, elegant, and user-friendly experience for showcasing interior design services.

## 🎨 Figma Design

<img width="4471" height="3448" alt="Decorio-Interior Design app" src="https://github.com/user-attachments/assets/7092cad5-b470-48a7-a0ac-19c3d8370e78" />


The complete UI/UX for this project was first designed in **Figma**, including:

- Home Page
- Rooms Section
- Services
- About Us
- Contact Page
- Login & Signup
- User Profile Dashboard
- Responsive Mobile Screens
